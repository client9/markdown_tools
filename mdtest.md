## can link text be split?

yes.

```markdown
[link 
  test](https://www.client9.com)
```

[link 
  test](https://www.client9.com)

## can link text and link url be on separate lines

No.

```markdown
[link test]
   (https://www.client9.com)
```

[link test]
   (https://www.client9.com)
